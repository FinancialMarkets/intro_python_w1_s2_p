Bond Cash Flows and Value


def create_bond_CFs(coupon, years):
'''
Create a list of bond cash flows.

:param float coupon: The bond's coupon rate.
:param int years: The number of years to maturity.
'''
    tmp = [coupon * 1000] * years
    tmp[years-1] = tmp[years-1] + 1000
    return tmp

Functions vs Methods

In documentation you may see the terms function and method. We won't make a distinction between the two in this course. The reasoning is:

  • Methods and functions are similar enough that the distinction is unimportant for now.
  • Methods have to do with classes, which we won't cover.

Python Object Oriented Programming (OOP)

OOP represents a more formal level of programming in Python. Learning it is one of the next steps to take after this course. However, here we'll show that you can be very productive, and use Python for many of your spreadsheet tasks, without delving into OOP.

  • That is, you don't have to learn the whole language structure before you can get the benefits of Python.
  • We won't create classes and will only refer to them if it aids in understanding the task.

List Comprehensions

A list comprehension is a convenient way to iterate over a list.


	      [2*x for x in range(5)]
	      ## returns the list [0, 2, 4, 6, 8]
	      ## note:
	      list(range(5))
	      ##returns [0, 1, 2, 3, 4]
